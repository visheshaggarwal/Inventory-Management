Inventory Management

Features Implemented

  • User-friendly application both for the customers as well as store admins
  • Registration of new products, customers, suppliers and inventories
  • Cart feature for customers
    • Searching all products of a particular brand Registered Suppliers can supply goods to the inventories


    Concepts of DBMS

  • SQL queries like insert, update and join
  • Conversion of ER Model to Relational Tables
  • Normalisation
    • Lock system to maintain consistency of system Used Semaphore Methodology to prevent deadlocks


    Technology Stack

  • Python-Flask Framework for web application
  • Flask-MySQL library for performing SQL queries
  • MySQL for maintaining the database
